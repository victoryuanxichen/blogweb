# Design System

## Color

使用 OKLCH 色彩。背景为带暖调的纸色，不使用纯白；文字为带绿褐倾向的深墨，不使用纯黑。主强调色为矿物绿，少量朱砂用于状态、选中和品牌印记。

## Typography

中文标题使用系统宋体/衬线栈，呈现安静的文学气质；正文使用系统无衬线，保持跨端可读。标题层级必须有明显尺寸和重量差。

## Components

卡片只用于真正需要边界的内容，例如订阅表单和阅读进度。文章列表采用横向节奏、图片裁切、分隔线和标签，不做重复等宽卡片网格。

## Motion

使用短促的 ease-out 过渡，不动画布局属性。交互反馈以颜色、阴影、轻微位移和内容切换为主。
